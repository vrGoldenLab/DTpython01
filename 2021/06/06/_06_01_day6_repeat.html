up&down 게임

  1. 1~100 까지 임의의 수를 정하고
  2. 정답을 맞출때까지 숫자를 입력한다
    • 내가 입력한 숫자가 정답보다 작으면 'Down' 출력
    • 내가 입력한 숫자가 정답보다 크면 'Up' 출력
  3. '정답'이라고 출력 후 종료
import random 
r_answer = random.randint(1,100)
m_answer = int(input('예상숫자:'))
while r_answer != m_answer:
    if r_answer < m_answer:
        print('Down')
    else:
        print('Up')
    m_answer = int(input('예상숫자:'))
print('정답')
